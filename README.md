# 🛡️ SOC Analyst Learning Repository (IBM QRadar & Splunk)

## 📖 Overview

Welcome to my SOC (Security Operations Center) learning repository.

This repository documents my journey of learning SOC operations, IBM QRadar, Splunk, Threat Intelligence, Incident Response, and Blue Team concepts through notes, labs, hands-on exercises, and practical examples.
